Sony

PSX

DuckStation is by far the best option, and the only you should consider. It's very fast, easy, and accurate. It also has many graphical enhancement options.
All PSX emulators need BIOS files.

PS2

PCSX2 is in the same situation as DuckStation, don't use anything else.
All PS2 emulators need BIOS files.

PS3

RPCS3 is your only option. It's still very experimental and requires a modern (2021+) high-end PC. I'd only recommend it for more advanced users.
RPCS3 requires PS3 firmware files which can be obtained fron Sony's PS3 update webpage.

PSP

PPSSPP is the only option you should really consider, very polished and easy to use. Does not require BIOS files and is easy to run.

PS Vita/TV

Vita3K is your only option, should be light enough of hardware, but is still very experimental. I'd only recommend this one to advanced users.
