Nintendo Consoles

Something to note:

Anytime I refer to "modern hardware" when talking about emulators for newer systems, assume I'm talking about a system that has an Intel i5-7600k or Ryzen 3600 CPU, and GTX 960 or RX 570 GPU.

NES

FCEUX,
Nestopia UE, and
Mesen are all great options,
I'd recommend FCEUX, but really any will do, experiment and see which one you like the most! Interestingly, Mesen supports HD texture packs for NES games, kinda neat if you're into that.

SNES

bsnes is by far the best and most accurate, but can be kind of demanding on hardware.
Snes9x is also very accurate, but much more friendly to older/lower power hardware, use this one on your Raspberry Pi!

N64

RMG, use this, it's easy and accurate enough. Alternatively,
Simple64 is supposedly pretty good, haven't used it as much.
RMG and Simple64 both have the same core, that being Mupen64Plus-Next, so they should run games near identically, and it really comes down to which UI you prefer. Feel free to try both!

GameCube/Wii

Dolphin is your only option, runs well.
The reason Dolphin emulates both GameCube and Wii is because architecturally the Wii is like, literally an overclocked GameCube with more RAM.

Wii U

Cemu is your only option, runs well on modern hardware, and is fairly impressive.

Switch

Yuzu is easier to setup, while
Ryujinx boasts higher game compatibility, and slight better performance, however both run great on modern hardwar.
Switch emulators need a system firmware dump to run.

GB/GBC

SameBoy is what I've always used, it's simple and easy to setup. Highly accurate.

GBA

mGBA is the only good option, easy enough to setup, decent accuracy.

DS

DS emulation isn't very polished yet, but out of all there is,
MelonDS is defintely one of the most polished options around right now.
DeSmuME is also a pretty solid choice, but I personally prefer MelonDS.

3DS

Citra is your only choice, still kind of experimental, but polished enough. Runs best on modern hardware.

If you want to emulate a more obscure Nintendo system such as the Virtual Boy, I'd suggest researching on the Emulation General Wiki.
